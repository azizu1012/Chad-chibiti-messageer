## Cài đặt

1. Clone repository:
   ```bash
   git clone https://github.com/azizu1012/Chad-chibiti-messageer.git
   cd Chad-chibiti-messageer
   ```
2. Cài đặt các dependencies (nếu có requirements.txt):
   ```bash
   pip install -r requirements.txt
   ```
